Data Structures and Algorithms - Course Syllabus

Module 1: Introduction and Analysis
• Introduction to Algorithms

• Asymptotic Notation (Big O, Omega, Theta)

Module 6: Trees

Module 11: Dynamic Programming

Module 12: Greedy Algorithms

Module 13: String Algorithms

Module 14: Advanced Topics
